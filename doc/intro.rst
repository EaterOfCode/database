Introduction
============

Ark Database is an efficient library to work with databases.

What's inside
-------------

- Connection manager
- Query builder
- Model
- Model factory

Supported Drivers
-----------------

- Mysql
- Sqlite
- Postgresql

Requirements
------------

- PHP5.4+
- PDO

Installation
------------

It's recommended to install with composer:

::

    composer require ark/database
